Week 4 · Lesson 07

CSS Basics — Selectors, Colours, Text

Week 4 · 75 minutes · Year 10 Industrial Technology — Multimedia

IND5-2IND5-3IND5-4

Learning intentions

  • I can link an external CSS stylesheet to an HTML file.
  • I can use element, class, and id selectors.
  • I can change colour, background, and text styles with CSS.

Success criteria

  • My HTML has <link rel="stylesheet" href="styles.css">.
  • My CSS file uses at least 3 selector types (element, class, id).
  • My page has a chosen colour palette — not black text on white default.

Do Now · 5 min

Open your most recent HTML page. Right now it probably looks like a 1995 website — plain text, blue links, no style. That is about to change.

In your head: imagine your page with:

  • A soft background colour
  • A different font for headings
  • Larger, more breathing-room text

I Do · ~10 min Teacher demonstration

Teacher creates a styles.css file in the same folder and links it:

<!-- inside head of HTML -->
<link rel="stylesheet" href="styles.css">

Then writes first CSS rules:

/* Element selector - affects all p tags */
p {
  color: #333;
  font-size: 18px;
  line-height: 1.6;
}

/* Class selector - only elements with class="highlight" */
.highlight {
  background-color: yellow;
  padding: 4px 8px;
}

/* ID selector - only the one element with id="main-title" */
#main-title {
  color: #1f3a68;
  font-size: 48px;
}

Apply in HTML:

<h1 id="main-title">My Website</h1>
<p class="highlight">This paragraph is highlighted.</p>
<p>This paragraph is not.</p>
Three selector types:
  • p — element selector (all p tags)
  • .name — class selector (elements with that class)
  • #name — id selector (the ONE element with that id)

We Do · ~15 min Guided build

Together, style a sample paragraph block with:

  1. Dark grey background on the body
  2. Off-white text colour
  3. A heading in one accent colour (your choice from the palette)
  4. Change the font using font-family: Georgia, serif;
body {
  background-color: #1a202c;
  color: #f7fafc;
  font-family: Georgia, serif;
  padding: 2rem;
}

h1 {
  color: #f59e0b;
}

You Do · ~35 min Independent practice

Open your L04 semantic HTML file. Create styles.css in the same folder. Link it.

Apply at least 8 CSS rules including:

  1. A body background colour
  2. A text colour on the body
  3. A font-family (try Georgia, Arial, or Verdana)
  4. Heading styles for h1, h2
  5. Link colour and hover colour
  6. A class selector (e.g. .intro) styled differently
  7. An id selector (e.g. #profile-photo) styled differently
  8. At least one rule for ul or li

Differentiation

Support

Provided HTML and an empty styles.css with commented TODOs. Student fills in colour values and font choices.

Core

Complete the 8-rule stylesheet from scratch.

Extension

Research and use one CSS pseudo-class (like :hover, :first-child, :nth-child(even)). Explain in a comment what it does.
