Week 4 · Lesson 08

The Box Model — Padding, Margin, Border

Week 4 · 75 minutes · Year 10 Industrial Technology — Multimedia

IND5-2IND5-3

Learning intentions

  • I can describe the box model (content, padding, border, margin).
  • I can use padding and margin to create breathing room.
  • I can inspect the box model of any element in DevTools.

Success criteria

  • My page uses both padding AND margin somewhere intentionally.
  • At least one element has a visible border.
  • I can open DevTools and see the coloured box model layers.

Do Now · 5 min

Every element on a webpage — every single one — is a box. Including the text you are reading right now. Including the image above. Including the button you will click.

Open any website in DevTools, click the Select Element arrow (top-left of DevTools), hover any text or image. See the coloured layers around it? Those are the box model.

I Do · ~10 min Teacher demonstration

Teacher opens DevTools on a sample page and explains the layers from inside out:

+----------------- MARGIN (outside) -----------------+
|                                                     |
|   +------------- BORDER ---------------+             |
|   |                                     |            |
|   |  +------ PADDING (inside) ----+     |            |
|   |  |                             |    |            |
|   |  |       CONTENT               |    |            |
|   |  |   (text, image, etc)        |    |            |
|   |  +-----------------------------+    |            |
|   +-------------------------------------+            |
|                                                     |
+-----------------------------------------------------+

In CSS:

.card {
  padding: 20px;        /* space INSIDE the border */
  border: 2px solid #1f3a68;
  margin: 40px;         /* space OUTSIDE the border */
  background-color: white;
}
Padding vs margin — the one rule that confuses everyone:
  • Padding = space between content and border (inside the element)
  • Margin = space between this element and the next one (outside the element)

Teacher shows the shorthand:

padding: 10px;                  /* all 4 sides */
padding: 10px 20px;             /* top/bottom, left/right */
padding: 10px 20px 30px 40px;   /* top, right, bottom, left (clockwise) */

We Do · ~15 min Guided build

Build a "card" together:

<article class="card">
  <h3>My first card</h3>
  <p>This is the card content.</p>
</article>
.card {
  background: white;
  padding: 24px;
  border: 1px solid #e4e7ec;
  border-radius: 10px;
  margin-bottom: 20px;
  max-width: 400px;
}

Now duplicate the article 3 times. See how the margin between cards works?

You Do · ~35 min Independent practice

Go back to your styled page from L07 and add the box model to it:

  1. Add padding inside your body (try 2rem or 32px)
  2. Add a max-width to your main content area so it doesn't stretch too wide on a big screen
  3. Turn one element (a section, a heading, an article) into a "card" with background, padding, border, and margin
  4. Use border-radius to round corners
  5. Inspect in DevTools — see the coloured box model layers

Differentiation

Support

Provided HTML with a card structure. Student fills in padding, border, and margin values until it looks good.

Core

Complete the task from scratch on own file.

Extension

Add box-sizing: border-box; to a universal selector and explain in a comment how it changes the box model maths.

Exit Ticket · 5 min

Exit ticket

1. Draw and label the 4 layers of the box model in order from inside to out:

Content → __________ → __________ → __________

2. Which property creates space INSIDE an element? ___________

3. Which property creates space OUTSIDE an element? ___________
