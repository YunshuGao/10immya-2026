Week 9 · Lesson 18

Contact Forms and Footer

Week 9 · 75 minutes · Year 10 Industrial Technology — Multimedia

IND5-2IND5-3IND5-5IND5-7

Learning intentions

  • I can build an accessible HTML form.
  • I can use HTML5 input types and validation.
  • I can connect a form to Formspree for a real submission.

Success criteria

  • My portfolio has a working contact form.
  • Every input has a label element linked by for and id.
  • Submitting the form sends a real email (via Formspree free tier).

Do Now · 5 min

Fill out one good form (e.g. your school enrolment) and one annoying form. What made one smooth and the other frustrating?

Common issues: unclear labels, no error messages, no keyboard support, tiny touch targets, no way to recover if you mess up.

I Do · ~10 min Teacher demonstration

Teacher builds an accessible form:

<form action="https://formspree.io/f/YOUR_ID" method="POST">
  <div class="field">
    <label for="name">Your name</label>
    <input type="text" id="name" name="name" required>
  </div>

  <div class="field">
    <label for="email">Email</label>
    <input type="email" id="email" name="email" required>
  </div>

  <div class="field">
    <label for="message">Message</label>
    <textarea id="message" name="message" rows="5" required></textarea>
  </div>

  <button type="submit" class="btn">Send message</button>
</form>
Accessible form rules:
  • Every input has a <label for="..."> linked to the input's id
  • Use the right type: email, tel, url, number (triggers right keyboard on mobile)
  • Use required for required fields (free HTML5 validation)
  • Don't hide labels — visible labels help everyone

Then setup Formspree (free tier, 50 submissions/month):

  1. Go to formspree.io and create a free account
  2. Create a form. Get your endpoint URL
  3. Paste the URL into your form's action attribute

We Do · ~15 min Guided build

Together, style the form to match portfolio colours:

.field { margin-bottom: 1rem; }
.field label { display: block; font-weight: 600; margin-bottom: 0.3rem; }
.field input, .field textarea {
  width: 100%;
  padding: 0.7rem;
  border: 2px solid #e4e7ec;
  border-radius: 6px;
  font: inherit;
}
.field input:focus, .field textarea:focus {
  outline: none;
  border-color: #1f3a68;
}

You Do · ~35 min Independent practice

Add a contact section (or a dedicated contact.html page) to your portfolio:

  1. Build the form with name, email, message fields
  2. Every field has a label, correct input type, and required where appropriate
  3. Sign up for Formspree if you want a real working form
  4. OR use mailto: action if you prefer: <form action="mailto:you@school.nsw.edu.au" method="POST">
  5. Build a proper footer: your name, year, copyright, links to projects
  6. Test the form submits something

Differentiation

Support

Pre-built form HTML provided. Student swaps in Formspree URL or mailto.

Core

Build and style form from scratch.

Extension

Add HTML5 pattern validation (e.g. phone number regex) and :invalid CSS styling for user feedback.
